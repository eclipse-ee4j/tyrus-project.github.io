The JSR 356 Reference Implementation

This page provides links to download the binary and source distributions of the Reference Implementation (RI) for JSR 356: Java API for WebSocket , which represents a subset of the Tyrus project.

To utilize Tyrus distributions, the recommended approach to getting started and using WebSocket, please goto the Tyrus project page and click on the "Start", "Download" or "Download Now" buttons.
